# Enterprise Performance Dashboard

## Project Overview
Built an authentication-enabled analytics dashboard using Streamlit that helps analyze advertising campaign performance across multiple dimensions such as platform performance, audience behavior, ROI, budget optimization, device insights, and trend analysis.

Users can sign up, log in securely, access dashboard insights, and log out after usage.

---

## Problem Statement
Marketing teams often struggle to analyze campaign performance across multiple platforms and audience segments in one place.

This project solves that problem by creating a centralized dashboard that provides actionable insights for campaign optimization.

---

## Key Features
- User Signup/Login Authentication
- Password Hashing using hashlib
- Session Management
- Logout Functionality
- KPI Dashboard
- Platform Performance Analysis
- Audience Analysis
- Creative Performance Analysis
- Budget & ROI Analysis
- Device Insights
- Trend Analysis
- Dataset Explorer
- Advanced Insight Assistant
